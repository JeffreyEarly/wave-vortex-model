#  N0

buoyancy frequency of the no-motion density


---

## Description
Real valued property with no dimensions and units of $$rad s^{-1}$$.

## Discussion
The buoyancy frequency $$N_0$$ is defined as $$N_0\equiv sqrt{ - \frac{g}{\rho_0} \frac{\partial \rho_\textrm{nm}}{\partial z} }$$.
