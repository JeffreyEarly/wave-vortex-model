#  Qpm

Preconditioner for G, size(Q)=[Nj x Nk]. G*eta = etahat, (QG)*eta = Q*etahat, so etabar==Q*etahat.


---

## Description
Real valued property with dimensions $$(j,K2unique)$$ and no units.
